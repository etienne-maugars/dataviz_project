# Assignment 2 MVP - Streamlit App

This is a data visualization story on student insomnia and educational outcomes.

## Narrative Goal

The app communicates a coherent message:

- Stress and lifestyle factors are linked to sleep disruption.
- Sleep disruption is linked to concentration and assignment impacts.
- These impacts align with lower academic outcomes.

## Run Locally

From the `Assignment_2` folder:

```bash
pip install -r requirements.txt
streamlit run app.py
```
