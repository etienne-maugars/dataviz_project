# Assignment 2 MVP - Streamlit App

This is an MVP (beta release) for a data visualization story on student insomnia and educational outcomes.

## Narrative Goal
The app communicates a coherent message:

- Stress and lifestyle factors are linked to sleep disruption.
- Sleep disruption is linked to concentration and assignment impacts.
- These impacts align with lower academic outcomes.

## Run Locally
From the `Assignment_2` folder:

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Public Deployment (for peer feedback)
Use Streamlit Community Cloud:

1. Push `Assignment_2` to a GitHub repository.
2. Go to `share.streamlit.io` and connect your repo.
3. Set app entry point to `Assignment_2/app.py`.
4. Deploy and share the public URL.
